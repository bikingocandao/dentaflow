## 📋 INSTRUCCIONES DE USO

### Cómo personalizar cada prompt:
1. Reemplaza todo lo que está entre [CORCHETES] con los datos reales del negocio
2. Agrega todos los servicios con sus precios exactos
3. Define los horarios disponibles para citas
4. Agrega las preguntas frecuentes específicas del negocio

### Cómo conectarlo a la API:
```javascript
const Groq = require('groq-sdk');
const client = new Groq({ apiKey: process.env.GROQ_API_KEY });

const systemPrompt = `[PEGA AQUÍ EL PROMPT DEL PLAN ELEGIDO]`;

const response = await client.chat.completions.create({
  model: "llama-3.3-70b-versatile",
  max_tokens: 500,
  temperature: 0.7,
  messages: [
    { role: "system", content: systemPrompt },
    ...conversationHistory
  ]
});
```
